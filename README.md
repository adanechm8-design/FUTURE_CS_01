# 🔐 Cyber Security Internship – Task 1

## 📌 Project Title
Vulnerability Assessment using Nmap and OWASP ZAP

---

## 📖 Overview
This project is part of the Cyber Security Internship program. The objective is to perform a basic vulnerability assessment on a target system using industry-standard security tools and document the findings in a professional report.

---

## 🎯 Objectives
- Perform network scanning using Nmap  
- Identify open ports and exposed services  
- Perform web vulnerability scanning using OWASP ZAP  
- Analyze security risks and provide recommendations  
- Document findings in a structured report  

---

## 🛠️ Tools Used
- Nmap (Network Scanning Tool)  
- OWASP ZAP (Web Vulnerability Scanner)  
- Windows OS  
- Browser (Chrome/Edge)  

---

## 🎯 Target System
http://scanme.nmap.org

---

## 🔍 Key Findings

### Nmap Results:
- 22/tcp → SSH  
- 80/tcp → HTTP  
- 9929/tcp → nping-echo  
- 31337/tcp → Elite service  
- Some ports filtered (firewall enabled)

### OWASP ZAP Results:
- Informational security findings  
- Low-risk alerts detected  
- No critical vulnerabilities found (test environment)

---

## ⚠️ Risk Assessment
Overall risk level: **Low to Medium**

---

## 💡 Recommendations
- Disable unused services  
- Use SSH key-based authentication  
- Implement security headers (CSP, HSTS)  
- Regular vulnerability scanning  
- Strengthen firewall rules
